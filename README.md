# Programação Orientada a Objetos em Java

Repositório dedicado ao ensino de Programação Orientada a Objetos (POO) em Java, com foco em conceitos fundamentais, padrões de design e boas práticas de desenvolvimento.

## Descrição

Este repositório contém materiais didáticos, exemplos práticos e exercícios para aprendizado de POO em Java. O ambiente de desenvolvimento recomendado é o NetBeans.

## Estrutura do Repositório
poo-java/
├── aulas/ - Apresentações e materiais de aula 
├── ementa/ - Conteúdo programático e cronograma do curso 
├── exemplos/ - Exemplos de código comentados 
├── exercicios/ - Exercícios propostos com diferentes níveis de dificuldade 
└── README.md - Este arquivo


## Requisitos

- Java Development Kit (JDK) 11 ou superior
- NetBeans IDE 12 ou superior
- Git para versionamento de código

## Tópicos Abordados

- Fundamentos de Orientação a Objetos
- Classes e Objetos
- Herança e Polimorfismo
- Encapsulamento e Abstração
- Interfaces e Classes Abstratas
- Tratamento de Exceções
- Collections em Java
- Padrões de Projeto
- Boas Práticas de Código

## Como Utilizar Este Repositório

1. Clone o repositório em sua máquina local:
  ```bash
  git clone https://github.com/yurikomuta/poo-java.git
```

2. Abra o NetBeans e importe os projetos conforme necessário

3. Estude os exemplos antes de realizar os exercícios

4. Teste seu código regularmente e valide a saída esperada

## Configuração do Ambiente

### NetBeans

1. Abra o NetBeans
2. Acesse File > Open Project
3. Navegue até a pasta do repositório
4. Selecione o projeto desejado e abra

### Compilação e Execução

Para compilar e executar um projeto via linha de comando:

```bash
javac NomeDaClasse.java
java NomeDaClasse
```
